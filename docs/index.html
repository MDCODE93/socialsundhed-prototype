Social Sundhed Aarhus prototype

Demo med syntetiske data — kører lokalt i browseren via SQLite (WebAssembly). Ingen data forlader din maskine.

▲
Denne prototype kører på en relationel datamodel forbi SharePoint's 5.000-grænse.
Lige nu indlæst: … ledsagelser · … forløb · … borgere · … kontakter. Filtrering, sortering og rapportering kører i <100 ms — den samme datamængde i en SharePoint-liste ville ramme List View Threshold.
